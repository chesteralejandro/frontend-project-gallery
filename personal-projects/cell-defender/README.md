# Cell Defender

A grid-based browser game inspired by classic arcade mechanics where you play as a white blood cell defending the body from invading microorganisms.

<br />

## 🎮 Gameplay Overview

- You control a white blood cell
- Move around a tile-based map
- Collect medicine pills
- Avoid enemy microorganisms
- Clear all pills to win the game

## ✨ Features

### 🧍 Player Movement

- Grid-based movement system
- Smooth interpolation (no snapping)
- Direction buffering for responsive controls
- Boundary and wall collision-

### 🧱 Map System

- Tile-based layout
- Walls and walkable paths
- Easily extendable level design

### 💊 Collectibles (Pills)

- Pills are placed across the map
- Collected when player moves over them

### 👾 Enemy AI

- Grid-based movement (same system as player)
- Avoids walls
- Smart chasing behavior

## 💡 Inspiration

This project is inspired by classic arcade gameplay loops
