# Orbitarium

A visually interactive Solar System simulation built using pure HTML, CSS, and JavaScript — no 3D libraries required.

This project demonstrates how far you can push CSS animations and transforms to simulate orbital motion and planetary rotation.

<br />

## 🚀 Features

- ☀️ Realistic Sun with texture, glow, and fake 3D shading
- 🪐 Mercury orbit animation using pure CSS
- 🔄 Smooth orbital motion using transform and @keyframes
- 🎨 Texture-based planets (NASA-inspired assets)
- 🌌 Space-like UI with scalable structure

## 🎯 Goals of This Project

- Simulate orbital mechanics visually
